# Pinterest Audience + Packaging Rules

Use this as a companion rules doc when writing pins, reviewing product candidates, or tightening GPT prompts.

## Audience Read

Current working assumptions:

* audience skews female
* strongest age bands are 18–34
* traffic is overwhelmingly mobile
* visual-first behavior matters more than technical detail
* likely buyers respond to:
  * convenience
  * aesthetics
  * low-effort upgrades
  * budget-friendly glow-ups
  * outcome-driven hooks

## Core Pin Rule

Do not lead with the product.

Lead with:

* the result
* the problem solved
* the transformation
* the convenience win

Examples:

* weak:
  * `Gel Nail Kit`
  * `LED Patio Lights`
* stronger:
  * `Salon Nails at Home in 20 Minutes`
  * `This made my backyard look expensive`

## What To Optimize For

Each pin should feel:

* fast to understand
* easy to imagine
* easy to buy
* easy to act on

That usually means:

* fewer words
* clearer benefit
* one obvious focal image
* one obvious reason to click

## Best Language Patterns

Use these when true:

* no mess
* beginner-friendly
* easy
* fast
* at-home
* under $30
* worth it
* no experience needed
* instant upgrade
* no salon needed

## Best Pin Formats

For one product, try to get at least:

1. problem pin
2. aesthetic pin
3. list pin
4. transformation pin
5. how-to pin

Useful bonus format:

* impulse-buy / Amazon-find pin

## Visual Rules

* vertical 2:3
* mobile-first readability
* top = hook
* center = product or result
* bottom = benefit/support line
* minimal clutter
* no tiny text
* no burying the product

## What To Avoid

* generic product-only titles
* over-explaining
* feature/spec-heavy copy
* cluttered collages
* cute visuals with no clear outcome
* hooks that are not searchable

## Quick Pre-Post Checklist

Before posting, ask:

* is the outcome obvious in one second?
* is the title searchable?
* is the value clear on mobile?
* does this feel easy, fast, or useful?
* can this same product support 3+ more pins?

If not, tighten the framing before posting.
