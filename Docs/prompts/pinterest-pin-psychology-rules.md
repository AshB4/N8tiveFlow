# Pinterest Pin Psychology Rules

Use this as a clean strategy layer when writing Pinterest affiliate pins, evaluating product angles, or tightening GPT prompts.

## Core Shift

Do not frame the pin as:

* `here is a product`

Frame it as:

* `here is how life gets easier, prettier, cheaper, or faster`

The pin should feel like:

* a life upgrade
* a convenience win
* a small transformation
* a save-money or save-time shortcut

## Audience Lens

Current working assumptions:

* audience skews female
* strongest age band is 18–34
* behavior is mobile-first
* buyers are visual, trend-aware, and convenience-driven
* many purchase on quick emotional + practical logic, not deep technical comparison

Use this to improve relevance, not to write in stereotypes.

## Default Pin Priorities

Pins should usually communicate one of these fast:

* saves time
* saves money
* looks better
* feels easier
* reduces effort
* gives a better result at home

## Hook Structures

Good default headline structures:

1. Pain -> solution
   * `Stop Paying $60 for Salon Nails`

2. Low-effort win
   * `5-Minute Nails at Home`

3. Outcome
   * `Salon Nails Without the Salon`

4. Social-proof style
   * `Everyone Is Switching to This`

5. Price/value trigger
   * `Under $25 and Actually Worth It`

These are templates, not mandatory formulas. Keep the wording honest.

## Impulse Triggers

Useful when true:

* Amazon find
* under $30
* worth every penny
* beginner-friendly
* no mess
* easy at-home version
* no experience needed
* instant upgrade

Do not stack all of them into one pin. Pick one clear value.

## Best Product Framing

Move from:

* feature list
* product display
* technical explanation

Toward:

* finished result
* transformation
* lifestyle payoff
* emotional + practical outcome

Examples:

* weak:
  * `LED Patio Lights 50ft Waterproof`
* stronger:
  * `This made my backyard feel like summer`

* weak:
  * `Gel Nail Kit Review`
* stronger:
  * `No salon needed for this look`

## Strong Pin Types Per Product

For each product, aim for:

1. problem pin
2. transformation pin
3. aesthetic pin
4. list pin
5. trend pin

Optional bonus:

* process pin
* impulse-buy pin

## Design Implications

Because traffic is mobile-first:

* large readable text
* short hook
* one focal object
* clear contrast
* no weak low-contrast text
* no cluttered layout

Good rule:

* if the benefit is not obvious in one second, the pin is too weak

## What To Avoid

* generic product-only pins
* technical/spec-heavy angles
* cluttered collages
* cute visuals with no clear value
* exaggerated hype with no searchable meaning
* writing to a caricature instead of a real buyer

## Final Check

Before posting, ask:

* is the benefit obvious instantly?
* is the result clearer than the product itself?
* does this feel easy, useful, or desirable?
* is the title readable and searchable?
* could this same product support 3+ more strong pins?

If not, the framing still needs work.
