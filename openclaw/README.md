# OpenClaw

OpenClaw is the discovery and decision layer for N8tiveFlow.

Use this folder for:
- monetization logic
- keyword and angle discovery
- seasonal planning
- winner tracking
- structured output definitions

Do not wire this into runtime app code until the decision system is stable.

Relationship to N8tiveFlow:
- OpenClaw finds what is worth making
- N8tiveFlow schedules, routes, and posts it
