# KEYWORDS — Traffic Engine

## Core Rule

Keywords determine:
- whether content is found
- who finds it
- whether they are ready to click or buy

Prioritize:
- clear intent
- specificity
- repeatability

Avoid:
- vague, broad keywords with no action intent

Rule:
- keyword quality matters more than content volume
- one strong keyword cluster beats ten weak generic ideas

## 1. High Intent Keywords (top priority)

These drive clicks + conversions.

Patterns:
- best X for Y
- top X for Y
- X under $X
- cheap X for Y
- X that works
- X for beginners
- X for small spaces
- X for [specific problem]

Examples:
- best desk setup for small spaces
- cheap kitchen gadgets that work
- self care ideas for burnout
- beginner hydroponics setup

Use these first.

Best for:
- Amazon
- owned products with direct problem-solution fit
- comparison and roundup content

## 2. Problem-Based Keywords

These convert well because the user already knows the pain.

Patterns:
- how to fix X
- how to improve X
- how to organize X
- how to clean X
- how to start X
- how to stop X

Examples:
- how to fix messy desk
- how to organize small kitchen
- how to start hydroponics at home

These map directly to product solutions.

Best for:
- owned products that solve a specific pain
- affiliate products with a simple utility case

## 3. Idea / List Keywords (Pinterest core)

High volume and highly repeatable.

Patterns:
- X ideas
- X inspiration
- X setup ideas
- X aesthetic ideas
- X hacks
- X essentials

Examples:
- desk setup ideas
- self care ideas
- cozy room ideas
- goblin core aesthetic ideas

Great for scaling content, but pair them with stronger angles.

## 4. Buyer Curiosity Keywords

Middle intent, strong CTR if paired with proof.

Patterns:
- X worth it
- is X good
- X review
- X vs Y
- X alternatives
- X you did not know you needed

Examples:
- standing desk worth it
- amazon gadgets you did not know you needed

Good bridge between traffic and conversion.

Rule:
- curiosity keywords need a product or comparison quickly
- do not let them become empty clickbait

## 5. Low Intent Keywords (use carefully)

These bring traffic, not always money.

Patterns:
- aesthetic X
- cute X
- vibe X
- inspo X

Examples:
- cute desk setup
- aesthetic room ideas

Use only when:
- paired with stronger angles
- used for reach
- tied to a visual-first product lane

## 6. Long-Tail Keyword Rule

Always prefer:

- `desk setup ideas for small spaces` over `desk setup`
- `self care ideas for burnout` over `self care`

Long-tail usually means:
- less competition
- clearer intent
- better product fit

## 7. Keyword Expansion System

Take 1 keyword and generate useful variations.

Example:

Base:
- desk setup ideas

Expand:
- desk setup ideas for small spaces
- desk setup ideas for programmers
- desk setup ideas on a budget
- desk setup ideas aesthetic but functional

Expansion rule:
- vary audience
- vary budget
- vary problem
- vary use case
- vary environment

This creates scalable content without random drift.

## 8. Keyword + Angle Pairing Rule

Keywords alone do not convert.

Always combine:

- Keyword + Angle

Examples:
- `desk setup ideas` + `cheap upgrades`
- `self care ideas` + `actually works`
- `kitchen gadgets` + `under $20`

This is where conversion starts.

## 9. Visual Fit Filter (Pinterest critical)

Only use keywords that can be:
- shown visually
- understood instantly
- turned into a pin

Reject if:
- too abstract
- too text-heavy
- requires explanation

## 10. Category Buckets

Organize keywords into buckets so they map to products and boards.

### Productivity / Dev
- desk setup ideas
- coding setup
- productivity tools

### Self Care / Coloring
- self care ideas
- stress relief activities
- relaxing hobbies

### Home / DIY
- small space ideas
- home organization
- diy decor

### Affiliate / Amazon
- amazon finds
- cheap gadgets
- must have items

### Seasonal / Cute / Halloween
- kawaii halloween coloring
- spooky cute coloring book
- cute halloween activity book

## 11. Evergreen Rule

Prioritize keywords that:
- do not expire
- can be reused
- work year-round

Example:
- `desk setup ideas` = evergreen
- `spring desk setup 2026` = limited

Use evergreen as the base inventory layer.

## 12. Seasonal Layer (secondary)

Use when relevant:
- christmas gift ideas
- fall decor ideas
- back to school essentials
- halloween coloring ideas

Use for spikes, not as the core system.

## 13. Rejection Filters

Reject keywords if:
- too broad
- no buying or action intent
- not visual
- too competitive with no angle
- not mappable to a product

Accept keywords if:
- specific
- useful
- visual
- tied to a product
- expandable into multiple variants

## 14. Default Safe Keywords

If unsure, start with:
- `ideas for X`
- `best X for Y`
- `cheap X that works`
- `X for beginners`

These are consistently reliable.

## 15. Operational Rule

When choosing keywords:
- start from product fit
- confirm the intent level
- confirm the visual
- then expand into variants

Do not chase traffic-first keywords if they do not support monetization.
